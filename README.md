# ECommerce

Este projeto foi gerado usando [Angular CLI](https://github.com/angular/angular-cli) versão 22.0.4.

## Servidor de desenvolvimento

Para iniciar um servidor de desenvolvimento local, execute:
```bash
ng serve
```

Uma vez que o servidor estiver rodando, abra seu navegador e vá para`http://localhost:4200/`. O aplicativo vai recarregar automaticamente sempre que você modificar algum dos arquivos de origem.

## Estrutura de código

O Angular CLI inclui ferramentas poderosas de scaffolding de código. Para gerar um novo componente, execute:

```bash
ng generate component component-name
```

 Para uma lista completa dos esquemas disponíveis (como`components`, `directives`, ou `pipes`), correr:

```bash
ng generate --help
```

## Prédio

Para construir o projeto, execute:

```bash
ng build
```

Isso vai compilar seu projeto e armazenar os artefatos da build em`dist/` diretório. Por padrão, a compilação de produção otimiza seu aplicativo para desempenho e velocidade.

## Executando testes unitários

 Para executar testes de unidade com o[Vitest](https://vitest.dev/)executor de teste, use o seguinte comando: 

```bash
ng test
```

## Executando testes de ponta a ponta

Para testes de ponta a ponta (e2e), execute:

```bash
ng e2e
```

O Angular CLI não vem com um framework de teste de ponta a ponta por padrão. Você pode escolher um que se encaixe nas suas necessidades.
